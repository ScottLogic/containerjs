Window API

Try me

Overview

The Window API is used to create a new browser window.

Constructor

  new ssf.Window(url, name, features)

Creates a new window. If a window with the same name already exists, that window will be reused.

Note: Window resuse is platform dependant. The API is inconsistent between browsers and Electron only allows this for windows created via the built in window.open() method (child windows), not new BrowserWindow() that is used to implement this API.

Syntax

  var newWindow = new ssf.Window(url, name, features);

Parameters

url
URL to open in the window. Must be prefixed with the protocol (i.e. http://)
name
Name of the new window. Should not contain whitespace
features (optional)

Object to set features of the new window.

  • child: boolean - make the new window a child of the current window (non browsers only).

Return Value

window
A reference to the new window.

Methods

  • getCurrentWindowId() (static) - Get the current window id as a string
